# Digital Marketing Assessment

## Overview

Welcome to the digital marketing assessment for IAI. This assessment is designed to evaluate your skills in HTML and email marketing. Your task is to recreate the provided `sample_email.png` file in HTML.

## Instructions

1. **Recreate the Email**: Using HTML, recreate the email design shown in `sample_email.png`. Ensure that your HTML code is clean, well-structured, and follows best practices for email development.

2. **Responsive Design**: The email should be responsive and look good on both desktop and mobile devices. Use media queries and other responsive design techniques as necessary.

3. **Inline CSS**: For email compatibility, use inline CSS styles. Avoid using external stylesheets or embedded styles.

4. **Images and Assets**: Use placeholder images if necessary. Ensure that all images are properly referenced and displayed in the email.

5. **Accessibility**: Make sure the email is accessible. Use appropriate alt text for images and ensure that the content is readable by screen readers.

6. **Submission**: Once you have completed the task, submit your HTML file along with any assets (images, etc.) in a zip file. Name the zip file as `YourName_DigitalMarketingAssessment_IAI.zip`.

## Evaluation Criteria

- **Accuracy**: How closely your HTML email matches the provided `sample_email.png`.
- **Code Quality**: Cleanliness, readability, and structure of your HTML code.
- **Responsiveness**: How well the email adapts to different screen sizes.
- **Accessibility**: Use of alt text and other accessibility features.
- **Best Practices**: Adherence to email development best practices, including the use of inline CSS.
